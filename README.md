## NESTJS COMMUNICATION API

## Description

Manage Notifications with Microsoft SQL Server and sending it via Websocket or Email



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:development

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
# Specific Test
$ npx jest --testPathPattern=src/role/role.controller.spec.ts

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
